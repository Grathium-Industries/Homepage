BUNDLER

Bundler is a program designed to streamline the distrivution, editing and execution process. A "bundle.yaml" file is used to bundle the source, runtime environment, execution scripts, author, description & license all into one simple file to ship to other users and developers. This project was made to get rid of the need to create long installation & running manuals to your projects. Simply create your "bundle.yaml" file and Bundler will take care of the rest.
